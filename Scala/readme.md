# install scala

`sudo apt-get update`

`sudo apt install openjdk-17-jdk`

`java --version`

`wget https://downloads.lightbend.com/scala/2.13.4/scala-2.13.4.deb`

`sudo dpkg -i scala-2.13.4.deb`

`scala`
